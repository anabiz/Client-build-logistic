# Rider App

React Native mobile app for delivery riders built with Expo SDK 54.

## Features

- **Dashboard**: Performance metrics and delivery overview
- **Deliveries**: Active delivery management with POD capture
- **History**: Past deliveries with search and filters
- **Profile**: Rider information and settings

## Setup

```bash
npm install
npx expo start
```

## Usage

- Scan QR code with Expo Go app
- Login with rider credentials
- Manage deliveries and capture POD photos
- Track performance metrics

## Tech Stack

- Expo SDK 54
- React Native 0.76.1
- React 18.3.1
- Ionicons for UI elements
- expo-image-picker for camera

## Color Theme

Primary: #F35C7A (Pink/Coral)
